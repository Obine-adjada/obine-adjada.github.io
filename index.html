Projet 01 / 05

Analyseur Firmware IoT · TP-Link TL-WR841N

✓ Terminé · Septembre 2025
Problème
Des centaines de millions de routeurs tournent avec un firmware non audité. Le TP-Link TL-WR841N est déployé mondialement sans qu'aucun audit public de sécurité n'ait existé. Un routeur compromis entraîne interception réseau totale, empoisonnement DNS et persistance de l'attaquant.
Solution
Reverse engineering complet : extraction du firmware avec binwalk, décompilation assembleur MIPS dans Ghidra sur 15+ fonctions, cartographie des 14 endpoints HTTP, et un framework d'exploitation Python. Résultat : 3 déclarations de CVE à TP-Link.
Résultats
  • Injection de commandes via /cgi-bin/luci : exécution de code à distance (CVSS 8.2)
  • Contournement d'authentification via /www/auth.lua : accès admin sans identifiants (CVSS 7.8)
  • Divulgation d'informations via /api/config : dump de configuration sans authentification (CVSS 7.5)
  • 15+ fonctions MIPS décompilées · 14 endpoints HTTP cartographiés
  • 5 scripts Python d'automatisation pour l'analyse et le fuzzing

Projet 02 / 05

Proxy IA Sécurisé · Protection LLM en Entreprise

✓ Terminé · Déc. 2025 – Fév. 2026
Problème
Quand les entreprises déploient des API LLM, un angle mort émerge : injection de prompts, jailbreaks et exfiltration silencieuse de données personnelles, numéros IBAN et clés API. Aucune protection native ne s'interpose entre l'utilisateur et le modèle.
Solution
Un proxy transparent interceptant tout le trafic entre utilisateurs et API LLM. Détecte et bloque 15+ types d'attaques avec >95 % de précision, masque automatiquement les données sensibles et produit une timeline forensique dans ELK, le tout sous 50 ms.
Fonctionnalités clés
  • Détection de 6 variantes d'injection de prompts, jailbreaks, SQLi/XSS dans les prompts
  • Masquage automatique des données personnelles / IBAN / clés API : les données n'atteignent jamais le LLM
  • Tableau de bord ELK : timeline forensique, géolocalisation des attaques, historique des alertes
  • Latence inférieure à 50 ms, totalement transparent pour les utilisateurs finaux
  • 400+ requêtes/sec validées en tests de charge niveau production

Projet 03 / 05

Détecteur d'Attaques d'Ingénierie Sociale en Temps Réel

⚡ En développement
Problème
Plus de 90 % des violations débutent par de l'ingénierie sociale. Les entreprises dépensent des millions en pare-feu mais restent sans défense face à un email de phishing bien conçu. Les outils existants détectent des patterns connus : ils ne comprennent pas les dynamiques de manipulation psychologique.
Solution
Un pipeline NLP détectant l'ingénierie sociale en temps réel sur email, messagerie et voix. Basé sur les 6 principes d'influence de Cialdini avec explicabilité par principe et actions de réponse automatisées au-delà de la simple détection.
Fonctionnalités clés
  • Couverture multi-vecteurs : email, messagerie instantanée, appels vocaux (Whisper ASR)
  • Scoring Cialdini 6 principes avec sortie d'explicabilité par principe
  • Pipeline temps réel via FastAPI + Celery + file de tâches Redis
  • Réponse automatisée : quarantaine, alerte, blocage proportionnel au score de menace
  • Mapping MITRE ATT&CK : T1566 Phishing, T1598 Gather Victim Info

Projet 05 / 05

Auditeur de Sécurité des Extensions Navigateur

✓ Terminé · Mai – Juil. 2025
Problème
Les extensions navigateur s'exécutent avec des privilèges élevés sur chaque page : elles peuvent lire silencieusement les mots de passe, intercepter toutes les requêtes HTTP et exfiltrer des données. Les équipes IT n'ont aucun outil automatisé pour auditer des milliers d'extensions sur un parc d'entreprise.
Solution
Un agent multiplateforme auditant les extensions Chrome, Firefox et Edge via une analyse AST approfondie (50+ patterns malveillants), croisement VirusTotal, scoring de risque pondéré 0-100 et désactivation automatique. Validé contre une vraie extension malveillante sur 500 postes d'entreprise.
Capacités de détection
  • Parsing AST (esprima) : obfuscation eval, payloads Base64, détournement de session, XSS
  • Croisement de hashs VirusTotal API sur tous les fichiers d'extension
  • Scoring pondéré 0-100 avec décomposition d'explicabilité par pattern
  • Désactivation automatique des extensions dépassant le seuil critique
  • "Ad Block Wonder" : 200 détections, risque 95/100, 500 postes, violation RGPD >100 000 € évitée
